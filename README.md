# CipherClaw 


* `A simple script to Encode and Decode text in Blank Format !`


**Credits- https://github.com/SpookySec

## Usage
Clone it by
```bash
git clone https://github.com/anubhavanonymous/CipherClaw
```
Now, Go to the CipherClaw Directory
```bash
cd CipherClaw
```
Run it 
```bash
bash cipherclaw.sh
```
